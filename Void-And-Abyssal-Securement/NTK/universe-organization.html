Universe Organization

Atoms exist within Universes.

Universes exist within Multiverses. An infinite amount of Universes can exist within a Multiverse.

Multiverses exist within Hyperverses. An infinite number of Multiverses can exist within a Hyperverse.

Hyperverses exist within Omniverses. An infinite number of Hyperverses can exist within an Omniverse.

Omniverses exist within Ultraverses. An infinite number of Omniverses can exist within an Ultraverse.

Ultraverses exist within the Godsverse. Only 6 Ultraverses can exist within the Godsverse.

Verses are considered extra dimensional. If it were to be rendered within the 3rd Dimension, the X verse existing within the Y verse would look like atoms within it.

The Godsverse

Only 1 Godsverse can exist at any single time. Every 'Verse' within the Godsverse must have its core destroyed for the Godsverse to be destroyed. This however, can happen within the same instance. In order for a new Godsverse to form, the previous Godsverse must be destroyed. We are on Godsverse 6.
